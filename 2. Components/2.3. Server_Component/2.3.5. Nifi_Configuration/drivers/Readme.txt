ojdbc11-full.tar.gz - JDBC Thin Driver and Companion JARS
========================================================
This TAR archive (ojdbc11-full.tar.gz) contains the 23.26.2.0.0 release of the Oracle JDBC Thin driver(ojdbc11.jar), the Universal Connection Pool (ucp11.jar) and other companion JARs grouped by category. 

(1) ojdbc11.jar (7718137 bytes) - 
(SHA1 Checksum: 2462b603e435c9a2707f41a1896e5b5be6f36520)
Oracle JDBC Driver compatible with JDK8, JDK11, JDK12, JDK13, JDK14, and JDK15.
 
(2) ucp11.jar (1585050 bytes) - (SHA1 Checksum: a7a3de2e338f5a52f293c6f247ba8469732d9bcd)
Universal Connection Pool classes to be used with ojdbc11.jar -- for performance, scalability, high availability, sharded and multitenant databases.

(3) rsi.jar (260580 bytes) - (SHA1 Checksum: 43b0b8af0e6972f4d773201dbb96d2654afefad9)
Reactive Streams Ingestion (RSI) 

======================
Security Related JARs
======================
Java applications require some additional jars to use Oracle Wallets. 
You need to use all the three jars while using Oracle Wallets. 

(4) oraclepki.jar (548026 bytes) - (SHA1 Checksum: 7601c59bc06e583c240d1ef377588dec5a1a6173)
Additional jar required to access Oracle Wallets from Java
